## Usage

1. Clone the git repository

   ```bash
   git clone https://github.com/karanBRAVO/VidX/edit/main
   ```

2. Change the directory to the cloned folder

   ```bash
   cd VidX
   ```

3. Run the script

   ```bash
   npm run dev
   ```
